# 📚 Estud.AI

O Estud.AI é uma aplicação web desenvolvida para transformar PDFs de estudo em resumos inteligentes utilizando Inteligência Artificial.

O usuário envia um arquivo PDF e a IA gera automaticamente um resumo didático, claro e objetivo com base no conteúdo do documento.

---

# 🚀 Funcionalidades

✅ Upload de arquivos PDF  
✅ Extração automática de texto  
✅ Integração com IA (Gemini API)  
✅ Geração de resumos inteligentes  
✅ Interface moderna e responsiva  
✅ Processamento de documentos em tempo real  

---

# ⚙️ Como Executar o Projeto

## 1. Clone o repositório

```bash
git clone URL_DO_REPOSITORIO
```

---

## 2. Instale as dependências

```bash
cd backend
npm install
```

---

## 3. Configure a API Key

Crie um arquivo `.env` dentro da pasta backend:

```env
GEMINI_API_KEY=sua_chave
```

---

## 4. Execute o backend

```bash
node server.js
```

---

## 5. Abra o frontend

Abra o arquivo `index.html`
ou utilize a extensão Live Server no VS Code.
